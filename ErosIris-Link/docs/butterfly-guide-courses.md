# 小蝶新手教学维护说明

小蝶的核心定位是“操作教练”，不是自动修改系统的诊断助手。核心课程定义在
`js/guide.js` 的 `COURSES`，课程进度保存在 `localStorage`：
`signalpath.guideCoursesV1`。

## 三秒欢迎气泡

页面启动后，小蝶会从 `WELCOME_LINES` 随机选择一句欢迎词，在蝴蝶旁显示
`WELCOME_MS`（当前为 3000ms）后淡出。点击气泡会直接打开教学首页。
气泡会根据蝴蝶位于屏幕左侧或右侧自动换向，避免拖动位置后超出屏幕。

新增文案时保持简短、鼓励和行动导向，不应承诺软件目前不存在的功能。

## 五门基础教学

1. 音响数据：下载填写模板、导入、当前案例提取、模板库预览与备份。
2. 音响反推：进入反推、填写数量、选择型号、检查参数、确认并创建系统。
3. 线材填写：单条长度、分组批量、勾选批量、采购汇总。
4. 工程报告：选择章节、跳转预览、浅色打印版、生成 PDF。
5. Excel 导出：选择数据章节并导出 Excel 可打开的 CSV。

基础教学首页提供“申请免学”。确认后，尚未完成的五门基础课会记录在
`courseData.skipped`，课程卡显示“已免学 · 可回看”。免学不会删除课程、
修改工程数据或影响进阶教学；用户点击任意已免学课程仍会从第一步开始回看，
完成后该课程会从“免学”转为“已完成”。

## 第一主界面进阶教学

进阶课程与基础课程复用同一套进度、暂停恢复、高亮和定位机制：

1. `advanced-global`：欢迎页、主题、报告、快速布局、模板、配置、清空、快捷键、全局撤销重做。
2. `advanced-config-json`：完整配置 JSON、模板库 JSON、导入核对、命名和外部备份习惯。
3. `advanced-config-switch`：当前配置、切换、临时移除、恢复、删除和逐套导出。
4. `advanced-views`：设备连线、线材清单、接线教学、台内路由和条件出现的交换机路由。
5. `advanced-canvas`：智能连接、清线、清设备、功率报警、方向及两种对齐。
6. `advanced-viewport`：分台视图、全局定位、缩放、框图撤销重做与导出。
7. `advanced-inspector`：添加设备、设备栏、连接清单和两种排序。

配置教学必须保持以下数据边界准确：

- “导出当前配置”导出当前活动配置的完整 `Store.state`，其中包括模板库和预设。
- 一个当前配置 JSON 不包含全部配置槽；多套方案要逐套切换并分别导出。
- “导出模板库”只保存设备模板、快速预设、反推预设和台面模板，不保存项目设备与连线。
- 配置切换是整套状态替换，不是跨项目数据合并。
- 临时移除只隐藏，删除不可撤销；危险步骤只能高亮讲解。

清空数据、清线、清设备等危险操作必须使用 `manual: true`，只高亮讲解，
不能要求用户真实点击后才继续。

## 步骤字段

- `text`：当前步骤说明。
- `target`：要定位的元素，支持 `#id` 或 CSS selector。
- `callout`：显示在目标旁边的短提示。
- `wait: "click"`：用户真实点击目标后自动进入下一步。
- `wait: "change"` + `eventTarget`：文件或表单发生变化后自动进入下一步。
- `manual: true`：显示“完成这一步，继续”。
- `ensure`：断点恢复时自动打开模板、反推、线材或报告上下文。
- `choices`：分支选择，例如模板课程的“下载导入/当前案例提取”。

新增步骤时优先监听真实操作完成，不要使用固定时间假装用户已经完成。动态弹窗
重绘后必须重新解析 selector，不应缓存旧 DOM 节点。

## 拓展教学

拓展项定义在 `EXT_TIPS`。首页每次只展示一条，已看项目短期内不重复。
Dante 仅在存在交换机或多台调音台时进入候选，保持靠后。
