================================================
/audio/ 文件夹说明
================================================

当前结构（默认音源都在 imports/ 里，随网页一并部署）：

/audio/
    imports/
        music-1/    ← "音乐文件"轨的默认歌曲（可放多首，页面内可选）
        vocal/      ← 人声分轨
        guitar/     ← 吉他分轨
        keys/       ← 键盘分轨
        bass/       ← 贝斯分轨
        drums/      ← 鼓分轨（放入文件后即可替换内置合成鼓）
        manifest.json  ← 音频清单（供不带目录列表的服务器扫描用）
        README.txt     ← 分轨导入、批量换歌的详细说明

【替换 / 新增音频】
1. 把文件放进上面对应的乐器文件夹（文件名随意，中文也可以）。
2. 双击项目根目录的"更新音频清单.command"重新生成 manifest.json
   （用 python3 -m http.server 这类带目录列表的服务器时可省略这步）。
3. 刷新页面，点"扫描端口文件夹"或对应轨道的"扫描"。

【格式建议】
- 兼容性最稳：wav / mp3；flac / m4a / aac / ogg 等主流浏览器也能解码。
- "粉红噪声 / 正弦波"轨为页面内置发生器，不需要任何文件。

【打开方式】
不要直接双击 index.html（file:// 会被浏览器拦截音频加载），
请双击项目根目录的"启动本地服务器.command"。
