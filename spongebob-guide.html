1. 30 秒上手

  1. 点「选择音频文件」，导入 mp3 / wav / m4a / aac / flac。
  2. 点「播放」，先听原声。用「总效果：前 / 后」随时对比。
  3. 需要时先剪切，再依次调 Gate → EQ → Chorus → 混响。
  4. 不确定怎么调时，先保留 Chorus 默认参数，EQ 少量处理，混响从小开始。
  5. 切到「导出」，选 WAV 保质量，选 MP3 方便发送。

处理顺序

剪切→Gate→EQ→ Chorus→混响→导出

一次只调一个参数，每次都点「前 / 后」对比。变大不一定变好，听得出变化后往回收一点，通常更自然。

2. 剪切

功能怎么用数值变大数值变小
Start 起点保留模式下，设定从哪一秒开始。前面被删掉的内容更多。保留更多开头。
End 终点设定保留到哪一秒。保留的结尾更长。更早结束，导出更短。
删除中间区间删掉 Cut Start 到 Cut End，前后会拼接。两个边界间的被删区间随之改变。适合去掉口误、空白或中间一段。

剪切后先听一次接口，避免把字头、换气或尾音切断。

3. Gate 噪声门

Gate 主要压低停顿和空白段的底噪，不是 AI 人声分离，不能完全去掉持续的强风噪或环境声。

参数作用大了会怎样小了会怎样
Threshold 门限低于这个电平时，Gate 开始压低声音。更多弱声被判定为噪声，可能吃掉轻声、换气和字尾。只压非常小的声音，更自然，但底噪保留更多。
Reduction 压低量Gate 关闭时将声音压低多少 dB。空白更安静，但容易出现忽有忽无的“抽吸感”。变化温和自然，但降噪不明显。
Attack 开启声音超过门限后，Gate 恢复打开的速度。打开较慢，可能把字头和拨弦瞬态磨掉。打开更快，字头清晰，但尖锐噪声也更容易通过。
Release 关闭声音低于门限后，Gate 慢慢关闭的时间。尾音更自然，但噪声会多留一会儿。很快安静，但可能截断尾音，听起来生硬。

建议：先播放只有底噪的停顿段，从较低 Threshold 慢慢提高，到刚好压住底噪即停。

4. 主 EQ

EQ 用来减少不需要的频率、突出需要的频率。先减后加，通常比一味抬高更稳。

滤波器 / 参数作用大了会怎样小了会怎样
低切频率切掉设定频率以下的低频。风噪、震动和轰鸣更少；过大会让声音变薄。保留更多厚度和低频，也可能保留轰鸣。
低搁架 Gain整体增减某个频率以下的低频。更厚、更暖；过大会闷、浑、轰。更轻、更干净；过小会单薄。
钟型 Gain增加或减少一个中心频率周围。正值越大，该频段越突出，也越不自然。负值越小，该频段越凹；过度会描边、空洞。
Q 值控制钟型 EQ 影响的宽窄。Q 越大，范围越窄，适合找谐振；过大容易尖锐。Q 越小，范围越宽，音色调整更平滑；也更容易影响整体。
高搁架 Gain整体增减某个频率以上的高频。更亮、更通透；过大会刺、齿音重、噪声多。更暖更柔；过小会暗、没空气感。
高切频率切掉设定频率以上的高频。频率越大，保留的高频越多，更亮。频率越小，声音越暗，可压刺耳也可能失去细节。

快速找频率

20–60Hz 震动 / 风噪60–120Hz 重量 / 厚度 120–300Hz 温暖 / 浑浊300–800Hz 箱体 / 闷 1–2kHz 存在感 / 鼻音2–5kHz 清晰 / 刺耳 5–8kHz 亮度 / 齿音8–16kHz 空气感 / 底噪

找难听频点：选钟型、Q 调大、Gain 暂时抬高，慢慢扫频率；找到最难听的位置后，把 Gain 改为小幅削减，再把 Q 放宽一点。

5. Chorus —— 海绵宝宝音核心

Chorus 复制并轻微漂移声音，造成拉开、水波和卡通感。本页默认已启用。

参数作用大了会怎样小了会怎样
Length原声与效果声之间的基础延迟。声音拉开，漂浮 / 回声感更强；过大像双轨不齐。更紧、更像普通 Chorus；太小时海绵宝宝感减弱。
Voices叠加多少个调制延迟声部。更厚更宽，像多人 / 合唱；也更模糊。更单纯、原声更突出；1 声部最符合本预设。
Rate音高漂移来回波动的速度。波动更快，卡通、泡泡、电子感增强；过大会抖。波动慢，像海浪，更温和真实；太小不明显。
Pitch Fudge音高 / 延迟漂移的幅度。晃动、梦幻、卡通感更强；过大会跑调、不稳。音高更稳、更干净，但特效感变弱。
Wet MixChorus 效果声的音量。注意：-6dB 比 -14dB 更大。数值越接近 0dB，水波和电子感越明显，也更容易模糊。数值越负，效果越隐藏，更接近原声。
Dry Mix原声的音量。同样是越接近 0dB 越大。原声更明显，拨弦 / 字头更清楚。原声减少，效果占主导；太小会空和远。

想更“海绵宝宝”：优先小幅增加 Wet 或 Rate，不要一次同时拉大 Rate、Fudge 和 Wet，否则很快会变得抖、散、听不清。

6. 混响

混响位于 Chorus 之后，会把已经带水波感的声音放进一个虚拟空间。Hall 更大更自然，Plate 更紧更亮。

参数作用大了会怎样小了会怎样
Pre-Delay原声到混响出现前的等待时间。原声与混响分开，字头更清楚；过大像小回声。混响紧贴原声，更融合；太小容易糊。
Decay混响尾巴持续多久。空间大、梦幻，但容易堆叠浑浊。干净利落，但空间感减少。
Size虚拟空间的尺寸。更宽广，距离感增加；过大会远。更亲密紧凑，像小房间。
Low Cut只切混响声的低频。混响更薄更干净，减少轰鸣。保留更多低频包围，也更容易浑。
High Cut只切混响声的高频。频率越大，混响越亮、细节越多。频率越小，混响越暗、更靠后。
Bass Multi控制混响低频尾巴的长度。低频拖尾更长，温暖厚重；过大会轰。低频收得快，更清爽。
Wet混响声百分比。空间感更大，声音更远、更模糊。更干、更近，混响可能听不出。
Dry送到输出的原声百分比。原声更清楚靠前，总音量也可能增加。原声减少，混响更主导，容易远和空。

混响 EQ

它只修饰混响返回声，不改原音。混响轰：提高 Low Cut 或在 150–300Hz 轻削；混响刺：降低 High Cut 或在 4–8kHz 轻削。

7. 学会前 / 后对比

  • 「后」 = 正在听处理后；「前」 = 旁通该处理。点击不会从头播放。
  • 总效果用来判断整体是否真的更好；EQ / Chorus / 混响按钮用来判断单个环节。
  • 处理后如果只是“更响”，会容易被误认为“更好”。对比时尽量让前后总响度接近。
  • 能连续听出“后”比“前”更清楚、更有趣且不刺耳，才是有效调整。

8. 模板与导出

模板

调好后输入名称，点「保存当前为模板」。模板保存在当前浏览器；换设备、换浏览器或清除缓存后可能丢失。

导出格式

WAV
无压缩、质量高、文件大。适合继续剪辑、混音或存档。
MP3
文件小、方便微信发送。192kbps 够用，256 / 320kbps 细节更多但文件更大。

手机建议先处理 30–90 秒。导出时不要切出页面或锁屏；微信内无法保存时，用右上角菜单选「在浏览器打开」。

10. 常见问题

声音太抖、像跑调

先降 Pitch Fudge，再降 Rate，最后降 Wet。不要同时把三者调大。

声音太模糊、字听不清

降 Chorus Wet 或 Voices；降混响 Wet / Decay；必要时提高 Pre-Delay，让字头先出来。

声音太薄

检查主 EQ 低切是否太高、低频是否削太多；检查 Chorus Dry 是否太小。

空白段一开一关，像被切断

降低 Gate Threshold 或 Reduction，增加 Release。

混响轰、低频堆在一起

提高混响 Low Cut，降 Bass Multi、Decay 或 Wet。

点导出后等很久

先剪切到需要的片段，短片段优先。MP3 还需要额外编码，通常比 WAV 慢。
